# java_fish_2
## Описание приложения

Данное приложение представляет собой веб-приложение на Java (Spring Boot),
предназначенное для хранения и обработки информации о рыбаках и их улове.

В приложении используется база данных PostgreSQL, в которой хранятся:
- данные о пользователях (рыбаках);
- информация о пойманной рыбе (название, вес, длина).

Приложение и база данных запускаются в отдельных Docker-контейнерах
с использованием Docker Compose.
